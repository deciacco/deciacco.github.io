			$ignoredLocalExtensions
						(line 173)
		
	


	

Stores the extension of files to be ignored by the local analysis.


All files which have an extension in this array will be omited from uploading to the  FTP server. Can be replaced with a new array. By default it ignores files with an  .lnk extension.
The example below would ignore all link files as well as text files.
Ex. $object->ignoredLocalExtensions = array('lnk','txt');
